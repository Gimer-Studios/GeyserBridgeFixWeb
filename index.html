GeyserBridgeFix Documentation

Description

The GeyserBridgeFix disables the placing of blocks in midair. Removing the advantage that Bedrock players have over Java players when playing on Geyser servers. This can help prevent Bedrock players from having a bridging and building advantage over Java players.

Features

  • Prevents placing of blocks in midair.
  • Lightweight plugin.

Commands

No commands in this plugin.

Permissions

No permissions required.

Configuration

There is a 5 second warning when Bedrock Players place a block infront of them. This warning tells them that they are using a disabled building method. You can turn this off if wanted through the config.yml thats is generated on startup of the plugin. It's default in true but false allows you to disable to provide a better competitive experience.

Very simple for users.

Usage

Useful when trying to balance a Geyser server with both Bedrock and Java players.

Compatibility

This plugin is compatible with Spigot, Paper, Bukkit, and more.
